Isi Kesimpulan Tujuan dari menambahkan atribut target di setiap link adalah untuk memberikan instruksi kepada browser untuk membuka halaman yang terkait. "_blank" berguna untuk membuka link pada frame tab baru, "_self" berguna untuk membuka link pada frame yang sama, "_parent" berguna untuk membuka link pada frame induk, sedangka "_top" berguna untuk membuka link dengan ukuran jendela penuh.
